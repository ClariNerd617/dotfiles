# dotfiles

This is the home of my dotfiles. For now, I am managing them manually while I research the best way to automate the process.

## Tools

I use bash for my shell. Yes, I know it is basic, but it is what I know best. I am also using [neovim](https://neovim.io/) as my text editor. I am still learning how to use it, but I am enjoying it so far.

I am using Oh-My-Bash which is similar to Oh-My-Zsh except you don't have to install zsh. You can find it [here](https://ohmybash.nntoan.com/).

I am very opinionated, so I will list the reasons I use Bash instead of Zsh below:

- I am lazy

This concludes the reasons I use Bash instead of Zsh.

I am also using [LazyVim](https://www.lazyvim.org/) which is an opinionated set of plugins on top of neovim to make it easier to use, and also lets me flex on everyone else at work.

For my terminal, I use [Wezterm](https://wezfurlong.org/wezterm/). You can find the `.wezterm.lua` file in this repo under the wezterm folder.

I have a personal policy of making sure what I use is easy on the eyes, which is why I use the Catppuccin themes for all of my tools. The plugins are already setup, so no need to research how to install them.

More tools will be added as I expand this repo.

## Installation

To install these dotfiles, literally clone the repo, I don't have anything fancy yet. I will add a script to automate the process in the future.

If you know what dotfiles are and don't know how to clone a repo, I have many questions for you, starting with "What the hell are you doing here?"
